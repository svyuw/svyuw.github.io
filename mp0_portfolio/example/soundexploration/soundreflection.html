MP2 Reflection

Reflection on Sound Exploration Assignment

Overview

For this assignment, we made six sketches using P5.js to make art with code.

To start off, I took some time to experiment with the P5.js editor. I started with the static image and thought about what I would like to make -- I decided on a fun and bright static image of a person. With this sketch, I used a forloop to create the background. For my second sketch, I didn't really have a plan for what I wanted my sketch to look like, but I started with generating random shapes (ellipses, rectangles, and triangles) that would populate the background. I also had them be random colors. I wanted to have a centerpiece to the sketch, so I made a static heart in the middle of the canvas. With my third sketch, I felt like I wanted to carry on with the rainbow / heart theme that I had going, so I tried making a heart that changed colors. I first made a singular small heart through reusing the same code as in my second sketch. My next step was to make it loop through the colors of the rainbow. Then, I accidentally made the heart grow bigger -- which turned out to be an accident that I liked the result of. For my fourth sketch, I made an interactive sketch where the user can draw on the canvas and have rainbow circles appear. These circles are closer together based off of speed of the user's mouse, and fade after a couple of seconds. My fifth sketch consists of the user clicking the canvas to play the sound "Be Your Girl (Kaytranada Edition)" by Teedra Moses and KAYTRANADA. If the user hasn't clicked the canvas, there is just a pink screen that says "click me to party!". After the user has clicked on the canvas, the screen turns different colors based on the position of the mouse on the canvas. My final sketch consists of sound input that results in rainbow colored circles that are larger if the sound input is larger.

To break down my process a bit more, I will discuss sketch three, the infinite loop sketch. To start, I reused the heart from my previous sketch. Then, I made the heart grow and shrink by setting a min, max, and original size for the heart to loop through. Once the heart has reached its max size, it will start shrinking. Once the heart has reached its min size, the heart will start growing again. Here, you can see the stroke growing and shrinking.

I originally wanted to make six hearts appear on a grid with them all changing colors. However, after playing around with the code, I liked how smooth the gradient was when I made the max size larger than the canvas width and height. One problem with my code for this heart is that if the screen is bigger than the max size that I set, then the user will see the blank canvas behind the heart. If I had more time, I think that I would try and make it so that the max size automatically adjusts to the size of the screen.

Issues

Each sketch had its own struggles, but I think one of the sketches that had the most puzzling issue was sketch six. I had a hard time getting the sketches to actually respond to the microphone input. Sometimes the code would work and the sketches would react to the sound, but other times I would do the same things and the sketches wouldn't. I found it just takes a while for it to load, but this was hard to work with because I would have to wait so long in order to fix my issues within my code and see the changes that I had just made. I inspected my code and also just waited for the code to fully load. It also took me a while to figure out how to map the amplitude to the color of the circles, but after some searching I found a helpful resource (at the bottom of this page) I am pleased with how the final product turned out.
