Final Project Proposal

For this assignment, I will be extending my web extension project and combining it with Unsplash API and Removal.AI to make random images of cats bounce around the user's screen.



Prototype Version

For my prototype version of this evolved web extension, I will be focusing on the core functionality of using the API to generate the images and having them bounce around the user's page. I will hardcode specific filters such as breed and category and mainly focus on making the API work with the browser extension.



Final Version

For my final version of this web extension, I would like to give the user the option to select which type of image they would like to bounce around their screen (static/animated, category such as animals or nature, and the option to choose a random picture).
